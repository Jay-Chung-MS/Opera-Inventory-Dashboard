수요예측 안내

판매계획은 영업팀의 목표치라서 실적보다 항상 높습니다. 이걸 그대로 발주 기준으로 쓰면 과잉재고가 쌓여서, 출고 트렌드 기반 예측을 별도로 만들어 발주 기준으로 씁니다.

WMA3 = 최근 3개월 가중 평균 (최근월 가중치 3배). 예: 12월 1만 / 1월 1.6만 / 2월 1.3만 → (1만×1 + 1.6만×2 + 1.3만×3) ÷ 6 = 1.32만
ETS = 6개월 출고를 순서대로 보면서, 새 데이터 30% + 이전 예측 70%로 누적. 오래될수록 영향 감소. "안정적 예측"
월예측출고 = (WMA3 + ETS) ÷ 2. 발주할 때 이 숫자 기준으로 판단
CV% = 수요 변동성. 높을수록 들쭉날쭉 → 예측 신뢰↓, 안전재고 더 필요
편차 = (월예측출고 - 판매계획) ÷ 판매계획 × 100. 음수 = 목표가 현실보다 높음
FCST 정확도 = 총량기준(큰 SKU 영향 큼) + SKU평균(소량 SKU 포함, 더 보수적)
